# WireGuardCLI 🛡️

[![Rust](https://img.shields.io/badge/rust-1.72+-orange?logo=rust&logoColor=white)](https://www.rust-lang.org/)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Version](https://img.shields.io/badge/version-0.1.0-green)](#)

**WireGuardCLI** ist eine schlanke Command-Line-Interface für **WireGuard**, geschrieben in **Rust**, damit du **nicht jede einzelne Kommandozeile auswendig lernen musst**. Alles läuft fast wie von alleine.  

## Features

- Einfaches Erstellen, Aktivieren und Deaktivieren von WireGuard-Interfaces  
- Automatisches Generieren von Konfigurationen  
- Übersichtliche Statusanzeigen und Logs  
- Rust-basiert: schnell, sicher und zuverlässig
